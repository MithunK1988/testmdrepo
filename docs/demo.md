## Summing numbers

Build a list with the `to` method:

```scala mdoc
1.to(10)
```

Build a list with the `until` method:

```scala mdoc
1.until(10)
```

Use `map` to transform the list.

```scala mdoc
// "hello".matches("h.*".length)
```
